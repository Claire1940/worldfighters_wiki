## Por que um Código Falha (E Como Corrigir)

Mesmo com a lista certa de códigos, o resgate pode falhar por motivos técnicos. Aqui está uma matriz prática de solução de problemas.

| Problema | Causa Comum | Correção Rápida |
|---|---|---|
| **“Invalid Code”** | Erros de digitação, caixa errada, símbolo ausente | Digite de novo com cuidado; compare caractere por caractere |
| **“Already Redeemed”** | A conta já resgatou | Passe para o próximo código |
| **Nenhuma recompensa exibida** | Lag de interface ou atraso do servidor | Entre novamente no servidor e confira inventário/moeda |
| **Código funcionou para outros, não para você** | Código expirou globalmente | Tente os códigos mais novos primeiro |
| **Colar não é aceito** | Caracteres ocultos no texto copiado | Cole em um app de notas e depois copie o texto limpo |

### Estratégia de Repetição com Pontuação

Para entradas antigas ou transcritas de forma confusa, você pode tentar repetições controladas:

- Primeiro, tente o código **exato** da lista.
- Se falhar, tente uma versão sem pontuação no final.
- Se ainda falhar, teste variações de espaço/capitalização apenas uma ou duas vezes.
- Não faça spam de tentativas rapidamente; alguns jogos limitam entradas.

Esse processo ajuda com strings incertas sem ignorar como os sistemas de código normalmente funcionam.
